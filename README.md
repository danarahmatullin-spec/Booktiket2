# Weather Diary - Дневник погоды

**Автор:** Рахматуллин Даниель
**Дата создания:** Май 2026

## Описание программы

Weather Diary — это графическое приложение для ведения дневника погоды. Позволяет:
- Добавлять записи о погоде (дата, температура, описание, наличие осадков)
- Просматривать все записи в удобной таблице
- Фильтровать записи по дате и температуре
- Сохранять данные в JSON-файл и загружать из него
- Удалять ненужные записи

## Требования

- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, datetime, os)

## Как запустить приложение

1. Скачайте файл `weather_diary.py`
2. Откройте терминал/командную строку в папке с файлом
3. Выполните команду:
   python weather_diary.py

## Примеры использования
### Тест 1: Добавление записей
1. В поле "Дата" оставьте текущую дату или введите, например: 2026-05-05
2. В поле "Температура" введите: 18.5
3. В поле "Описание" введите: Солнечно, легкий ветер
4. Отметьте "Осадки" (если нужно)
5. Нажмите "Добавить запись"

### Тест 2: Фильтрация
 * По дате: Введите 2026-05-05 в поле фильтра даты и нажмите "Применить фильтр"
 * По температуре: Введите 10 в поле температуры фильтра, чтобы увидеть записи с температурой выше +10°C
 * Сброс фильтра: Нажмите "Сбросить фильтр"

### Тест 3: Сохранение и загрузка
1. Добавьте несколько записей
2. Нажмите "Сохранить в JSON" — создастся файл weather_diary.json
3. Закройте и снова откройте приложение (данные загрузятся автоматически)
4. Или нажмите "Загрузить из JSON" для ручной загрузки

### Тест 4: Проверка валидации
 * Попробуйте добавить запись с неверной датой (например, 32.13.2025)
 * Попробуйте добавить запись с текстом в поле температуры
 * Попробуйте добавить запись с пустым описанием
